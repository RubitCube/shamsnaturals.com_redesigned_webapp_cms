# Shams Naturals Frontend

React + TypeScript + Tailwind CSS frontend for Shams Naturals e-commerce website.

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and configure
3. Start development server: `npm run dev`
4. Build for production: `npm run build`

## Pages

- `/` - Homepage with banners, best products, new arrivals
- `/products` - All products with category filters
- `/products/:slug` - Product detail page
- `/new-arrivals` - New arrivals page
- `/about` - About us page
- `/dealers` - Dealer network with Google Maps
- `/contact` - Contact form with reCAPTCHA
- `/blog` - Blog listings
- `/blog/:slug` - Blog detail page

## Components

- `Navbar` - Navigation bar with mobile menu
- `Footer` - Footer with links and copyright
- `SocialIcons` - Floating social media icons
- `BannerCarousel` - Homepage banner carousel
- `ProductCard` - Product card component
- `SEOHead` - SEO meta tags component
